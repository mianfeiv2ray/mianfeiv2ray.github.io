Java字符串compareTo()方法示例。

  1. 首页
  2. 新闻资讯
  3. 正文

String类的compareTo（）方法。此方法按字典顺序比较两个字符串。比较是基于字符串中每个字符的Unicode值。在字典上比较此String对象表示的字符序列与自变量字符串表示的字符序列。该方法返回

  • 如果当前String对象在字典上在参数字符串之前，则为负整数。

  • 如果当前String对象在字典上跟随该参数，则为正整数

  • 字符串相等时为true。

示例

import java.lang.*; public class StringDemo {    public static void main(String[] args) {       String str1 = "tutorials", str2 = "point";       //比较str1和str2-       int retval = str1.compareTo(str2);        //打印比较的返回值       if (retval < 0) {          System.out.println("str1 is greater than str2");       } else if (retval == 0) {          System.out.println("str1 is equal to str2");       } else {          System.out.println("str1 is less than str2");       }    } }

输出结果

str1 is less than str2
